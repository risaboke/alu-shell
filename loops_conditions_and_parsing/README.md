# Shell, loops, conditions and parsing

This project covers Bash scripting fundamentals: for/while/until loops, if/elif/else and case statements, file test operators, and parsing data with read, IFS, and awk.

## Tasks

- 1-for_best_school: Displays "Best School" 10 times using a for loop
- 2-while_best_school: Displays "Best School" 10 times using a while loop
- 3-until_best_school: Displays "Best School" 10 times using an until loop
- 4-if_9_say_hi: Displays "Best School" 10 times, with "Hi" on the 9th iteration, using a while loop and an if statement
- - 5-4_bad_luck_8_is_your_chance: Loops from 1 to 10, printing "Best School", swapping in "bad luck" on the 4th iteration and "good luck" on the 8th, using if/elif/else
- - 6-superstitious_numbers: Loops from 1 to 20, printing each number, and adds a superstition message ("bad luck from China" on 4, "bad luck from Japan" on 9, "bad luck from Italy" on 17) using a case statement
- 7-clock: Displays the time, looping hours from 0 to 12 and minutes from 1 to 59 for each hour, using nested while loops
- 8-for_ls: Lists current directory contents, trimmed after the first dash, using a for loop
- 9-to_file_or_not_to_file: Checks existence, emptiness, and type of a file named school, using if and else statements
- 10-fizzbuzz: Classic FizzBuzz from 1 to 100, using a while loop and if/elif/else
- 11-read_and_cut: Extracts username, UID, and home directory from /etc/passwd, using a while loop with read
- 12-tell_the_story_of_passwd: Generates a narrative for each user in /etc/passwd, using a while loop with IFS and read
- 13-lets_parse_apache_logs: Extracts visitor IP and HTTP status code from Apache logs, using awk
- 14-dig_the-data: Groups and counts visitor IP/status code occurrences, sorted from the greatest to the lowest number of occurrences, using awk
